# Weather Forecast App ☁️🌦️

A clean and simple weather app built using **React Native**.

## Features

- 🔍 City Search
- 📍 Location-based Forecast
- 📅 3-Day Forecast
- 🌡️ Temp, Humidity, Wind, Sunrise/Sunset
